This plugin allows you to import articles and other content written by Brandpoint or any other source using the Brandpoint custom feed format. 

== Description ==

Articles are imported into a specified category and assigned to a specified author.  Currently only one feed URL at a time is supported.

Plugin uses AJAX so you'll need JavaScript enabled in order to use it.

== Installation ==

1. Upload 'BrandpointFeedImporter.zip' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress

Plugin's page appears under the Settings menu as Brandpoint Feed Importer
